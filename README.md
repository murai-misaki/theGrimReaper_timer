## サービス名
### the Grim Reaper Timer　〜1時間座り続けるごとにあなたの寿命を22分戴きます〜


## サービス概要
- 「座りすぎで縮んだ寿命を可視化する、座りすぎ防止Webタイマーアプリ」です。
- 1日のうちに長時間、座って作業をしている人に、
30分に１回立ち上がることを習慣化させ、1時間座り続けると寿命が22分縮まることを防ぎ、1日8時間以上座ることによって高まる罹患リスクや死亡リスクを軽減させます。
- このアプリは、1時間座り続けると22分死神が寿命を縮めるという世界観で、座りすぎの恐ろしさを表現しているホラーテイストのタイマーアプリです。


## メインのターゲットユーザー
- (デスクワーカーや受験生、プログラミングの勉強や資格勉強をしている人など)1日のうちに長時間、座って作業をしている人


## ユーザーが抱える課題
- 1時間座り続けると寿命が22分縮む
- 1日8時間以上座っている人は、罹患リスクや死亡リスクが高まる
	(→座りすぎは肥満、がん、うつ病、糖尿病、高血圧、心疾患、脳梗塞など将来的にさまざまな病気を誘発すると考えられている)
- 座りすぎによるリスクを知らずに、長時間座りっぱなしの人は多い
	(→日本人の座る時間は平均8~9時間と、世界の中でも日本人はダントツで座っている時間が長い)
- ついつい長時間座り続けて、作業をしてしまう
- 座りっぱなしは水分不足になりやすく、これが血液の循環を悪くすることもある
- 身体活動不足


## 解決方法
- 座りすぎによるリスクを伝える
- 30分ごとに通知を行う
   (→一回に座ったまま過ごす時間が30分を超えないようにすること。これで座りすぎのリスクはかなり減らせます。)
- 1日ごとの、座っている時間と1時間座り続けたことで縮んだ寿命を記録し、直近1週間分をグラフで可視化する
- 立ち上がらずに座り続けて1時間が経過すると、その日の座りすぎによって縮んだ寿命がリアルタイムで、タイマー使用中に表示される
- 1時間座り続けたことで縮んだ寿命の合計時間を知れる
- 上記を通して座りすぎが健康に及ぼすリスクについて知り、ユーザー自身が危機感を持つことで、30分に1回立ち上がり3分のブレイクタイムを設けることを習慣化できるようになる
- 1日に合計8時間座っている場合、通知によって注意喚起をする
- ブレイクタイム時に水分補給を促すメッセージを表示する
- ブレイクタイム時に、体を動かす4ジャンル(全身バランスよく・肩こり肩痛予防・腰痛膝痛予防・下肢筋力アップ)計13種類の1分程度の軽い運動を紹介する
- ブレイクタイム時に行った、1日ごとの身体活動量を記録し、直近1週間分をグラフで可視化する


## アプリ機能
#### 座りすぎによるリスクを手軽に効果的な対策ができる機能
- 30分ごとに立ち上がりのお知らせを致します
    - ''30分に一度3分、立って動く''
       一回に座ったまま過ごす時間が30分を超えないようにすること。これで座りすぎのリスクはかなり減らせます。
- お好きなお知らせ方法を選択できます(※随時変更可能)
    - 音でお知らせする、サウンド通知
    - 視覚でお知らせする、プッシュ通知
- 危機感を持つことで、30分に1回の立ち上がりを怠らないように促す機能
    - 立ち上がらずに座り続けて1時間が経過すると、座りすぎによって縮んだ寿命がリアルタイムで表示
    - 1日ごとの、座っている時間と1時間座り続けたことで縮んだ寿命を記録し、直近1週間分をグラフ化
    - 1時間座り続けたことで縮んだ寿命の合計時間を知れる (※実装中)
- 1日8時間以上座ることによって高まる罹患リスクや死亡リスク、身体活動不足、水分不足も対策できる機能が備わっています
    - 1日に8時間以上座り続けている場合に注意喚起を行い、罹患リスクや死亡リスクが高まるのを防ぐ機能
    - ブレイクタイム時に体を動かす4ジャンル計13種類、1分程度の軽い運動をご紹介
    - 30分に1回水分補給を促すメッセージを表示
    - ブレイクタイム時に行った1日ごとの身体活動量を記録し、直近1週間分をグラフ化
       (→1日の身体活動量が10分(1回1分の運動を10回)増加すると、3.2%のリスク※が減少することが明らかになっています。)  
        ※ 死亡、生活習慣病・がん・ロコモ・認知症の発症(厚生労働省 : 健康づくりのための身体活動量2013)

## なぜこのサービスを作りたいのか？
- 『1時間座り続けると、寿命が22分縮む』という事実を知って、とてもゾッとしたのがきっかけです。
- 座りすぎのリスクを知った後でも、ついつい長時間座りっぱなしでプログラミングをする生活を続けるままで、このまま改善しないと、より積み重なり、長期化して、エンジニア転職後も仕事や勉強を頑張れば頑張るほど、座りすぎによる寿命の短縮や死亡リスクが高まることに非常に危機感を抱きました。
そのため、単なるタイマーアプリではなく、もし立ち上がることを怠れば座りすぎで縮んだ寿命を可視化でき、その危機感から実際に立ち上がらざるを得なくなるような、「30分に1回立ち上がること」を確実に習慣化できる、特別なタイマーアプリが欲しかったです。
- また、日本人は世界でも座っている時間が最も長いとされ、座りすぎによるリスクについて十分に認識していない方が多いのではないかと考えました。
このアプリのを通じて、座りすぎの恐ろしさを伝え、無意識に座り続けることで寿命を縮めてしまうリスクを防ぐための対策アプリとして、日常的に利用していただけることを目指しています。

## スタンディングデスクなどの物理的な解決策にはない当アプリの利点
- スタンディングデスクがなくても現在のデスク環境のままで、座りすぎによるリスクについて知ったその日から、手軽に効果的な対策を行えることが利点だと考えます。
- 2017年時点では、日本においてスタンディングデスクの普及率は10％以下でした。
現在では、普及率は上昇していると考えられますが、それでも実際にスタンディングデスクを所有して使用している人々はあまり多くないと思われます。
そのため、スタンディングデスクを自分の作業環境に導入する場合には、デスクを購入する手間や費用がかかるため、多くの人々が面倒だと感じてやめてしまうことがあると考えられます。
また私自身、学生時代から座って作業することに慣れているため、立って作業するよりも座って作業する方が楽で、じっくりと集中できると感じています。
また、ネット上でも、スタンディングデスクを使って腰や膝が痛くなったり、立ち続けることでの疲労が主な原因で、使用をやめる人々が多いです。
しかし、当アプリを利用することで、座りすぎによるリスクを知ったその日から、現在の慣れ親しんだデスク環境のままで、手軽に効果的な対策を行うことができます。これは、当アプリの大きな利点の一つと言えます。


## 類似したサービスと当サービスの相違点や特徴
- タイマー使用前に座りすぎによるリスクについて具体的に知れる
- 立ち上がらずに座り続けて1時間が経過すると、その日の座りすぎによって縮んだ寿命がリアルタイムで、タイマー使用中に表示される
- 1日ごとの、座っている時間と1時間座り続けたことで縮んだ寿命を記録し、直近1週間分をグラフ化
- 1時間座り続けたことで縮んだ自分の寿命の合計時間を知れる
→このように座りすぎによるリスクを知ってもらい、ユーザー自身が危機感を持つことで、30分に1回の立ち上がりを怠らないように促すことができます。
- 他には類を見ないホラーテイストのタイマーアプリでユーザーの興味を引き、1時間座り続けると22分死神が寿命を縮めるというアプリの世界観から座りすぎの恐ろしさを伝えています。
- ダウンロード不要ですぐに使用できること(座りすぎ防止に特化したタイマーアプリで、現在あるのはモバイルアプリのみ)
- ユーザーが好きなタイマーの通知方法を選択できること(プッシュ通知 or サウンド通知)
    →私は、作業中に突然アラーム音が鳴るのに驚いてしまい作業中のストレスになるため、既存の代替可能なサービスの利用を控えていました。私のようなアラーム音が苦手な人や、オフィスや図書館などでアラーム音を出すことができない環境でも使用できることは、他のタイマーアプリとの大きな違いだと思います。
- 座りすぎによるリスク対策として、単に30分に1度立ち上がることを促すだけでなく、1日8時間以上座ることによって高まる罹患リスクや死亡リスク、身体活動不足、水分不足などの、座りすぎによって引き起こされる「1時間座り続けると寿命が22分縮む」以外の問題についても解決できる機能が備わっていること
    - 30分に1回ブレイクタイム時に水分補給を促すメッセージを表示する
    - 1日に8時間以上座り続ける場合に通知によって注意喚起を行い、罹患リスクや死亡リスクが高まるのを防ぐためのサポートをする機能
    - ブレイクタイム時に体を動かす4ジャンル計13種類の1分程度の軽い運動を紹介する、身体活動不足を解消するための機能
    - ブレイクタイム時に行った、1日ごとの身体活動量を記録し、直近1週間分をグラフ化
